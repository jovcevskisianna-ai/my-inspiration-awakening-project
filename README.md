Inspirational Awakening

General Information:

Inspirational Awakening is a musical composition that combines music and technology to help musicians who are facing a creative block. This composition introduces musicians to a diverse range of instruments and sounds, guiding them through uplifting, deep and dynamic sequences that represent different emotional tones. The piece showcases how rhythm, layering and tone contrasts and inspires various pathways for musical exploration.

Technologies Used:

Tunepad and Python was used to code and compose musical sequences. Digital audio tools were utilised for layering and refining the different instruments. Git and Github has been used for collaboration between all aspects. 

Project Status:

The composition, presentation and critical reflection has all been completed.

Features:

It introduces musicians to a diverse range od instruments and tones structured into four distinct musical phases. 

Room for improvement:

My musical composition could've incorporated some cultural aspects, expanded transitions and intergrated user interaction to personalise musical pathways.
